# 위드 물티슈 랜딩페이지 PRD

> 기준 버전: `index-v7.1.html`  
> 작성일: 2026-07-02  
> 상태: Draft  
> 한 줄 목표: 식당·카페·매장 사장님이 "위드는 직접 만드는 제조사라 싸고, 물티슈가 끊기지 않겠다"를 이해한 뒤 전화·카톡·견적으로 문의하게 만든다.

## 1. 배경

위드 물티슈 랜딩은 일반 소비자용 브랜드 소개 페이지가 아니라, 업소용 물티슈를 반복 구매하는 사장님을 위한 문의 전환 페이지다. 기존 방향은 대량 단가와 정기 납품을 중심으로 잡았지만, v7.1에서는 메시지가 더 좁혀졌다.

- 위드는 도매상이 아니라 직접 만드는 제조사다.
- 중간 유통이 없어서 단가 경쟁력이 있다.
- 직접 재고와 생산을 관리하므로 공급이 끊기지 않는다는 신뢰를 줄 수 있다.
- 방문자가 긴 폼을 채우기보다 전화·카톡으로 빠르게 문의하는 흐름이 더 자연스럽다.

따라서 v7.1의 제품 방향은 "가격표 공개형 도매몰"이 아니라 "직접 제조 신뢰를 확인하고 바로 연락하는 B2B 문의 랜딩"이다.

## 2. 문제 정의

식당·카페·매장 사장님은 물티슈를 살 때 예쁜 브랜드 이야기보다 아래 정보를 빠르게 확인하고 싶다.

- 우리 매장에 맞는 제품 형태가 있는가?
- 너무 얇거나 품질이 들쭉날쭉하지 않은가?
- 단가가 납득 가능한가?
- 주문했을 때 안 끊기고 받을 수 있는가?
- 세금계산서, 샘플, 상호 인쇄 같은 사업자 거래가 가능한가?

현재 해결해야 할 핵심 문제는 방문자가 "여기 믿고 문의해도 되겠다"까지 도달하기 전에 정보가 흐리거나, 가짜처럼 보이는 placeholder 때문에 신뢰가 깨질 수 있다는 점이다.

## 3. 목표

### 3.1 사용자 목표

- 제품군과 규격을 보고 내 매장에 맞는 물티슈를 대략 고른다.
- 직접 생산, 재고, 납품 안정성을 보고 공급 신뢰를 판단한다.
- 전화·카톡·견적 폼 중 편한 방식으로 문의한다.

### 3.2 비즈니스 목표

- 업소용 물티슈 대량·반복 구매 문의를 만든다.
- 단가표 공개 없이도 "제조사 직접 거래라 싸다"는 인식을 남긴다.
- 실제 상담 전에 품목, 지역, 월 사용량, 연락처를 확보한다.

### 3.3 비목표

- 온라인 결제 기능을 제공하지 않는다.
- 전체 제품 카탈로그를 상세 쇼핑몰처럼 제공하지 않는다.
- 확정되지 않은 가격, 재구매율, 납품 정책을 실제값처럼 단정하지 않는다.
- v7.3 실패 실험의 구조를 이어받지 않는다.

## 4. 타깃 사용자

### 4.1 1차 타깃

식당·카페·매장 사장님 또는 구매 담당자.

- 모바일로 빠르게 본다.
- 통화와 카톡 문의에 익숙하다.
- 장당 단가와 재고 안정성에 민감하다.
- 세금계산서 발행과 사업자 거래를 중요하게 본다.

### 4.2 주요 사용 상황

- 기존 물티슈 거래처 단가가 비싸다고 느낄 때
- 물티슈가 자주 떨어져 안정적인 납품처를 찾을 때
- 상호 인쇄 물티슈를 새로 만들거나 바꾸려 할 때
- 통화 전 제품 종류와 규격을 먼저 확인하고 싶을 때

## 5. 핵심 가치 제안

### 5.1 메인 메시지

"직접 만드는 제조사라 더 싸고, 안 끊깁니다."

### 5.2 보조 메시지

- 11년간 업소용 물티슈 직접 제조
- 누적 납품 경험
- 세금계산서 발행
- 위생용품 신고필
- 무료 샘플 가능
- 상호·로고 인쇄 가능

### 5.3 신뢰의 논리

1. 왜 싼가: 중간 유통이 없다.
2. 왜 안 끊기나: 직접 재고를 쌓아두고 공급한다.
3. 왜 품질이 일정한가: 원단·평량·매수를 직접 관리한다.
4. 왜 믿을 수 있나: 11년 동안 같은 자리에서 제조했다.

### 6.1 전환 지표

- 전화 버튼 클릭 수

- 상세 견적 폼 제출 수

### 6.2 품질 지표

- 필수 정보가 포함된 문의 비율: 지역, 월 사용량, 연락처

### 6.3 콘텐츠 완성 지표

- 생산 라인, 창고, 원단·품질, 공장 전경 사진 교체 완료

- 재구매율, 누적 납품, 거래처 수 등 숫자 검증 완료
- FAQ 정책 실제값 반영 완료
- 사업자 정보와 연락처 실제값 반영 완료

## 7. 페이지 구조

### 7.1 상단 내비게이션

목적: 방문자가 주요 신뢰 근거와 문의 지점으로 바로 이동하게 한다.

필수 항목:

- 로고: 위드 / 업소용 물티슈
- 링크: 직접 생산, 후기, 제품, 자주 묻는 질문, 견적문의
- CTA: 견적 받기

### 7.2 히어로

목적: 첫 화면에서 업소용 물티슈 제조사, 납품 안정성, 문의 행동을 즉시 전달한다.

필수 내용:

- 대상: 식당·카페·매장 업소용 물티슈
- 헤드라인: 11년간 4만 건+, 한 번도 안 끊겼습니다.
- 보조 문장: 도매상을 거치지 않는 직접 제조사라 더 싸고 안 끊긴다.
- 신뢰 메타: 세금계산서 발행, 위생용품 신고필, 무료 샘플
- 1순위 CTA: 전화
- 2순위 CTA: 카톡
- 3순위 CTA: 견적 남기기
- 콜백 안내: 바로 못 받아도 그날 안에 연락

확인 필요:

- 전화번호
- 카카오톡 채널 URL
- "그날 안에 연락" 약속 가능 여부
- 4만 건+ 근거와 표현 가능 여부

### 7.3 신뢰 숫자 바

목적: 직접 제조와 납품 경험을 숫자로 빠르게 증명한다.

필수 항목:

- 제조 연차
- 거래 업소 수
- 재구매율
- 누적 납품 건수

확인 필요:

- 재구매율 83%와 99% 중 실제 사용 가능한 값
- 거래처 수와 누적 납품 건수의 산정 기준
- 과장으로 보이지 않도록 기준 설명 필요 여부

### 7.4 직접 생산 시네마

목적: "직접 생산"을 말로만 주장하지 않고 화면 스크롤로 체감하게 한다.

구성:

- 스텝 1: 왜 싼가 / 중간 유통이 없습니다.
- 스텝 2: 왜 안 끊기나 / 11년간 결품 0.
- 스텝 3: 왜 품질이 일정한가 / 원단부터 직접 관리.
- 스텝 4: 왜 믿을 수 있나 / 11년, 같은 자리에서.

기능 요구:

- 데스크톱과 모바일에서 스크롤에 따라 단계가 전환된다.
- 모션 최소화 설정에서는 모든 단계가 정적으로 보인다.
- 사진 또는 영상이 없어도 텍스트는 읽혀야 한다.

필수 자산:

- 생산 라인 사진 또는 영상
- 창고 재고 사진
- 원단·품질 관리 사진
- 공장 전경 사진

### 7.5 후기

목적: "진짜 거래하는 사장님이 있다"는 사회적 증거를 준다.

필수 내용:

- 공급 안정성 후기
- 단가 경쟁력 후기
- 세금계산서 및 거래 편의 후기
- 규격 신뢰 후기

확인 필요:

- 실명 또는 익명 동의를 받은 실제 후기 2~4개
- 업종과 규모 표기 가능 여부

### 7.6 제품군

목적: 방문자가 자신의 사용 상황에 맞는 물티슈 형태를 고르게 한다.

제품 카드:

- 개별포장: 손님 테이블용, 배달 동봉
- 캡형: 카운터·매장 비치용
- 대용량 리필: 주방·청소용 대량 사용
- 대량 공급: 재고 기반 빠른 출고

필수 자산:

- 개별포장 실사진
- 캡형 실사진
- 대용량 리필 실사진
- 창고 또는 대량 출고 사진

### 7.7 규격 안내

목적: 상담 전에 원단, 평량, 매입 범위를 이해하게 한다.

필수 내용:

- 원단 종류: 메쉬, 헤링본, 엠보
- 전체 평량 범위: 45g~100g
- 원단별 지원 평량:
  - 메쉬: 45g, 50g, 55g, 70g, 100g
  - 헤링본: 45g, 80g
  - 엠보: 55g, 60g, 100g
- 1박스당 매입: 400, 600, 800, 1000, 1200

주의:

- 지원하지 않는 평량은 선택 가능한 것처럼 보이면 안 된다.
- 실제 생산 가능 범위가 바뀌면 표를 먼저 수정한다.

### 7.8 상호·로고 인쇄

목적: 주문제작 가능성을 보조 전환 포인트로 제공한다.

필수 내용:

- 가게 이름, 연락처, QR 인쇄 가능
- 견적에 함께 문의 가능

확인 필요:

- 제작 기간
- 초기 비용
- 최소 주문 수량
- 디자인 지원 범위

### 7.9 FAQ

목적: 전화 전 자주 묻는 질문을 줄인다.

필수 질문:

- 최소 주문 수량이 얼마인가요?
- 단가가 어떻게 되나요?
- 세금계산서 발행되나요?
- 샘플을 먼저 받아볼 수 있나요?
- 상호 인쇄는 기간이 얼마나 걸리나요?

확인 필요:

- 최소 주문 수량
- 배송비 정책
- 샘플 정책
- 상호 인쇄 기간과 비용

### 7.10 문의

목적: 전화 중심으로 빠르게 문의하게 하되, 통화가 어려운 사람도 남길 수 있게 한다.

우선순위:

1. 전화
2. 카카오톡 상담
3. 번호 남기기
4. 상세 견적 폼
5. 이메일

번호 남기기 필드:

- 전화번호

상세 견적 필드:

- 관심 품목: 개별포장, 캡형, 대용량 리필, 주문제작, 잘 모름
- 지역
- 월 예상 사용량
- 업종
- 연락처
- 추가 요청

필수 기능:

- 필수값 누락 시 화면 안에서 안내한다.
- 제출 후 접수 상태가 보인다.
- 실제 배포 전에는 전송 연동이 필요하다.

## 8. 콘텐츠 요구사항

### 8.1 반드시 실제값으로 교체할 항목

- 전화번호
- 카카오톡 채널
- 이메일
- 사업자 정보
- 대표자명
- 사업자등록번호
- 위생용품 신고번호
- 주소
- 거래처 수
- 재구매율
- 누적 납품 건수
- 결품 0 표현의 근거
- 최소 주문 수량
- 상호 인쇄 기간과 비용

### 8.2 반드시 실제 자산으로 교체할 항목

- 생산 라인
- 창고
- 원단·품질 관리
- 공장 전경
- 제품별 사진
- 실제 후기

### 8.3 카피 원칙

- 사장님이 바로 이해하는 짧은 문장으로 쓴다.
- 제조사 직접 거래, 공급 안정성, 규격 통제를 반복해서 각인한다.
- 확정되지 않은 수치와 정책은 실제값처럼 말하지 않는다.
- 감성적인 프리미엄 표현보다 거래 신뢰를 우선한다.

## 9. 기능 요구사항

### 9.1 반응형

- 모바일 우선으로 동작해야 한다.
- 하단 고정 전화·카톡 바는 모바일에서 보인다.
- 제품 카드와 규격 표는 모바일에서 세로로 무너지되 정보가 겹치면 안 된다.

### 9.2 접근성

- CTA와 폼 요소는 키보드로 접근 가능해야 한다.
- 관심 품목 선택 버튼은 선택 상태를 보조기술에 전달해야 한다.
- 모션 최소화 설정에서는 직접 생산 시네마가 정적 콘텐츠로 보여야 한다.
- 본문 대비는 연회색 배경에서도 충분해야 한다.

### 9.3 폼

- 번호 남기기와 상세 견적 폼은 각각 제출 상태를 보여준다.
- 배포 전 실제 접수 방식이 결정되어야 한다.
- 가능한 연동 후보:
  - 문자 알림
  - 카카오톡 채널
  - 이메일
  - Formspree 같은 외부 폼 서비스
  - 자체 서버 API

### 9.4 성능

- 실사진이 들어가도 첫 화면 로딩이 과하게 느려지면 안 된다.
- 시네마 섹션 이미지는 압축과 lazy loading을 적용한다.
- 외부 스크립트가 실패해도 핵심 문구와 문의 CTA는 보여야 한다.

## 10. 배포 전 수용 기준

- 첫 화면에서 업소용 물티슈, 직접 제조, 전화·카톡 문의가 5초 안에 이해된다.
- 전화 버튼은 실제 번호로 연결된다.
- 카카오톡 버튼은 실제 채널로 연결된다.
- 모든 placeholder 경고가 실제 콘텐츠로 교체되거나, 의도적으로 숨김 처리된다.
- 모바일에서 하단 CTA가 주요 콘텐츠를 가리지 않는다.
- 직접 생산 시네마가 데스크톱과 모바일에서 깨지지 않는다.
- 모션 최소화 설정에서 콘텐츠가 누락되지 않는다.
- 상세 견적 폼 제출 후 사용자가 접수 여부를 알 수 있다.
- 사업자 정보와 정책 문구가 실제값으로 들어가 있다.
