# 위드 랜딩(v7.1) — 콘텐츠 수집 숙제

> v7.1 화면의 ⚠ placeholder를 실제값으로 채우기. 수집 **동선별**로 묶음 — 한 번 움직일 때 몰아서.
> 확정 방향: 직접생산=척추 / 단가 비공개 / 4채널(전화1순위·카톡·폼·이메일) / 무전제 설계.

## 1. 📸 사진 촬영 (한 번 나갈 때 몰아서) — **v7.1 임팩트의 핵심**
직접생산 허브가 시네마(풀스크린 전환)라 **사진 퀄이 곧 설득력**. 가로 풀프레임, 텍스트가 위에 얹히니 좀 어두워도/여백 많아도 OK.

**허브 4컷** (순서 = 싸다→안끊김→품질→신뢰)
- [ ] ① **생산 라인** (가동 중 장면) — "직접 만든다" 증거
- [ ] ② **창고** (박스 가득 쌓인 재고) — "안 끊긴다" 증거
- [ ] ③ **원단·품질** (원단 롤 / 평량·매수 관리 장면) — "품질 일정" 증거
- [ ] ④ **공장 전경** (간판·외관, 11년 같은 자리 느낌) — "믿을 수 있다" 증거
- [ ] (욕심) ①~④를 **짧은 영상**으로도 — 있으면 v7.1 시네마 효과 극대화

**제품 갤러리 4컷**
- [ ] 개별포장 / 캡형 / 대용량 리필 / 창고·대량공급 장면

## 2. 📞 부모님 확인 (통화·방문 시 한 번에)
- [ ] **재구매율 정본** — 83% vs 99% 어느 게 맞나 (스토어 상세=99%, 랜딩 합의=83%). 과장 위험 없게 정확히.
- [ ] **콜백 시간** — "그날 안에" 실제 지킬 수 있나? 못 받으면 보통 언제/몇 시간 내 답하나 → 카피 "○○ 안에 연락" 확정용
- [ ] **대표 전화번호** — 랜딩엔 **아빠 직통** 맞나? (박스 인쇄 032=엄마 위드폰과 구분)
- [ ] **최소 주문** — "30박스 전후" 맞나
- [ ] **상호인쇄** — 기간 + 초기비용(동판비 1도18/2도36만 + 필름 33만 ≈ 51만)을 랜딩/상담에서 어떻게 노출할지
- [ ] **사업자 정보** — 상호·대표명·사업자등록번호·위생용품 신고번호·주소

## 3. 📝 후기 (진짜로 교체 — "반칙급 카드")
가짜 4개 → 실명/동의받은 진짜로. 도매 단골은 리뷰 안 하지만, **부탁하면 들어줄 10년 단골**이 있음(신규 셀러는 못 하는 카드).
- [ ] 동의 가능한 2~4곳: 한마디 + 업종/규모 (실명 or 익명동의)
