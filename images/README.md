# 위드 랜딩 — 이미지 폴더 안내

실제 사진을 여기 폴더별로 넣어주세요. v7.1 화면의 ⚠ placeholder 자리에 그대로 들어갑니다.

| 폴더 | 넣을 것 | v7.1에서 쓰이는 곳 |
|---|---|---|
| `production/` | **생산라인·창고·원단·공장전경** (직접 촬영한 실사진) | **직접생산 허브(척추)** 4컷 — 시네마 전환. ⚠ 가장 중요·실사진 권장(AI 금지) |
| `products/` | 제품 컷 (개별포장·캡형·대용량 리필·대량공급) | 제품 갤러리 4컷 |
| `detail/` | 상세페이지 캡처·기존 상세 이미지 | 참고/소재용 (필요 시 발췌해 랜딩에 재활용) |
| `brand/` | 로고·대표 이미지·현재 활용 중인 이미지 전부 | 톤앤매너/브랜드 키트 작업 소스 (다음 세션) |

## 권장 사항
- 가로(landscape) 풀프레임이면 허브·갤러리에 잘 맞음.
- 허브 사진은 텍스트가 위에 얹히니 좀 어둡거나 여백 많아도 OK.
- 영상(생산 b-roll)이 있으면 `production/`에 같이 — v7.1 시네마 효과 극대화.
- 파일명은 자유. 넣어두면 다음 세션에서 자리에 매핑해 교체.

## 기존 파일 (poc-*)
`poc-*.png`는 초기 PoC용 임시 이미지. 실사진 들어오면 교체·정리 예정.
