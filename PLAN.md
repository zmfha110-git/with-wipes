# 위드 물티슈 — 홈페이지 기획 문서

> 작성일 2026-06-17 · 상태: 컨셉 확정, 시안 착수 전
> 한 줄: **식당 사장님에게 "업소용 물티슈, 대량은 더 싸게·정기로 안 끊기게"를 큰 숫자로 들이밀고 견적 문의로 끌어오는 랜딩.**

---

## 1. 목적 (이 사이트가 해야 할 한 가지)

- **1차 목적**: 브랜드 소개 + **대량·정기납품 견적 문의 받기**.
- 직접 결제(이커머스) 없음 — 문의/견적이 전환 지점.
- "위드는 믿고 싸게, 안 끊기게 받는 업소용 물티슈 거래처" 라는 인식을 남긴다.

## 2. 타깃 독자

- **식당·카페·매장을 운영하는 자영업 사장님** (구매 결정자 = 사용자).
- 손님상에 깔리는 **업소용 물티슈**로 타깃을 좁힘. (유아용/소비자용 라인 아님 — 베베숲 "시그니처 위드"와 혼동되지 않게 명확히 분리.)
- 행동 특성: **모바일로 본다**, 원가에 민감하다, 전화·카톡을 편해한다, "떨어지면 장사 못 한다"는 재고 불안이 있다.

## 3. 포지셔닝 & 차별화

우선순위:
1. **대량 단가** — 수량 많을수록 싸진다 (가성비 증거).
2. **정기 납품(안 끊김)** — ★ 핵심 차별화. 시장에 거의 없는 빈자리.
3. **주문제작(상호·로고 인쇄)** — 작은 보조 옵션으로만.

**차별화 한 방 = "정기 납품".**
리서치 결과 이 시장은 대부분 단발 주문이고, 명시적 정기배송(구독형) 옵션은 드물다. → "주문 깜빡해서 떨어질 일 없게, 알아서 채워지는 납품"은 경쟁사가 잘 안 쓰는 각이라 먹힌다. 단가는 그 뒤를 받치는 증거.

> 가격만 내세우면 최저가 치킨게임. 정기납품은 "원가 항목"을 "관리 안 해도 되는 편함"으로 프레임을 바꿔 경쟁축을 옮긴다.

## 4. 톤 & 비주얼 방향

- **대담·가성비 임팩트**. 큰 숫자, 강한 대비, 직관적 (마트 전단의 명료함 + 정돈).
- 흰 바탕 + 강한 포인트색. 위생/신뢰 느낌 유지하되 감성 럭셔리는 배제.
- **모바일 우선** 반응형.
- 히어로 예시: `"장당 ◯◯원부터 · 월 정기납품"` 큰 숫자 + `[지금 견적받기]`.

## 5. 페이지 구성 (스크롤 순서)

| # | 섹션 | 핵심 내용 |
|---|---|---|
| 1 | **히어로** | "장당 ◯◯원부터 · 월 정기납품" 큰 숫자 + [지금 견적받기] / 보조: 전화·카톡 |
| 2 | **왜 위드** | 대량할인 · 정기배송(안 끊김) · 빠른 출고 — 3칸 |
| 3 | **제품군** | 개별포장 · 캡형 · 대용량 리필 (업소 용도별) |
| 4 | **단가 구간표** | 수량 많을수록 ↓ (가성비 증거) |
| 5 | **정기납품 작동방식** | 신청 → 주기 설정 → 자동 배송, 3스텝 |
| 6 | **주문제작(보조)** | "상호 인쇄도 됩니다" 한 섹션, 작게 |
| 7 | **신뢰 신호** | 위생용품 신고 · 거래처 수 · 생산/공급 역량 |
| 8 | **견적 문의 폼** | 업종 / 예상 수량 / 납품 주기 / 연락처 → 제출 |
| 9 | **푸터** | 전화 · 카톡 · 사업자 정보 |

## 6. 주 전환(CTA)

- 메인: **"지금 견적받기"** — 업종·예상수량·주기·연락처를 받는 폼.
- 보조: 전화 바로 걸기, 카카오 채널 상담 (모바일에서 특히).

## 7. 기본값 가정 (확정 전 임시)

- 단가는 **예시 숫자로 노출** + "정확한 건 수량별 견적" (대담 톤엔 숫자 노출이 맞음).
- 문의 폼은 **데모로 동작** → 실제 발송은 추후 이메일/카톡/Formspree 연동.
- 1차 시안은 **플레이스홀더 에셋**으로 제작 → 실제 자산 받으면 교체.

## 8. 채워야 할 실제 정보 (사용자 입력 대기)

1. 로고 · 브랜드 색 (있으면)
2. 대표 단가 예시 1~2개 (예: "1매 개별포장 장당 ◯◯원")
3. 취급 품목 (개별포장 / 캡형 / 대용량 중 뭐뭐?)
4. 신뢰 신호 (위생용품 신고 여부, 거래처 수, 전화번호)

## 부록 A. 시장 리서치 요약 (근거)

식당용 물티슈 도매 시장 웹 리서치 결과 (2026-06-17, 출처 확인됨):

- **사장님 구매 결정 순서**: ① 장당 단가 × 최소주문(MOQ) → ② 포장 형태·매수 → ③ 평량(촉감/안 찢어짐) → ④ 배송·재주문 편함 → ⑤ 위생인증(의외로 후순위, 헤드라인에 거의 안 씀).
- **가격 통념**: 무지 낱개 30~50원 / 캡형·대용량 매당 8~15원 / 커스텀 인쇄 33~50원+. (출처마다 편차 큼 — 범위로 봐야 함.)
- **규격 통념**: 낱개 1매형 박스당 500~1,000매, 캡형 70·100매, 대용량 400매·1,000매. 평량 35~55gsm가 식당용 흔한 구간, 원단은 스펀레이스 부직포.
- **주문제작(상호 인쇄)**: 이 시장의 사실상 표준 판촉물. "전단은 버려도 물티슈는 받아 쓴다 → 다 쓸 때까지 가게 이름 15~20회 노출." MOQ 보통 5,000매, 무료 로고 디자인이 거의 기본. → 단, 위드는 이걸 **보조**로만 두기로 함.
- **정기배송(구독형)**: 시장에 명시 옵션이 드묾 → **차별화 빈자리**.
- **경쟁사 헤드라인 패턴**: 도매몰은 "당일출고·대용량 가격·용도별(식당/카페/호텔)", 제조사는 "원스탑·자동화·품질테스트". 위생 인증을 메인 카피로 박은 곳은 거의 없음.
- **"위드 물티슈" 브랜드 특정 여부**: 독립 브랜드로 **특정 안 됨**. 동명은 베베숲 유아용 라인 "시그니처 위드"뿐 → 위드 물티슈는 외부 인지도가 거의 없으므로 포지셔닝을 새로 잡아도 무방.

> 주의: 위 수치는 공급사 상품 페이지 기반 역추정 + 통념이며 확정가 아님. 사장님 직접 후기(자영업 카페/블로그) 1차 자료는 추가 조사 여지 있음.
